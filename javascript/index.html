Porque Aprender a Programar?

Cada dia a tecnologia avança para uma nova descoberta, todos os dias novas empresas são criadas e novos negócios são iniciados e aprender a programar pode te colocar neste mundo louca em um local privilegiado, onde você pode criar o que as pessoas utilizam e até, querendo ou não, influenciar as pessoas para consumir os seus negócios e produtos.

Agora Porque Aprender JavaScript?

JavaScript é uma linguagem de programação que está cada vez mais forte no mercado, seja para criar aplicativos, web sites, aplicação em robótica e cada vez se expandindo para novas vertentes.

Tem emprego para isso?

Segundo pesquisas em 2015 o número de vagas abertas para a linguagem JavaScript ultrapassou 2000+ vagas, atualmente existem mais de 4000+ vagas em aberto para a linguagem JavaScript e a cada dia novas vagas surgem no mercado, espalhadas por todo o Brasil e mundo afora essa numeração aumenta de forma exponencial.

Eu preciso apenas criar programas?

Programar não serve apenas para criar programas, antigamente sim, porém hoje em dia cada vez mais outras áreas e segmentos do mercado buscam profissionais que tenham compreensão de linguagens de programação.

E quanto que eu vou ganhar se eu souber trabalhar com JavaScript?
